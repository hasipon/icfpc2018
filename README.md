# DiamondPrincess ICFPC2018

## Required

+ ctest
+ clang++3.8

## Build

```
cmake .
make
```

## Run

Run ctest with AI name regexp. See [AI](AI) directory.

```
ctest -R hasi21
```

Then, nbt files are generated in [out](out) directory.
